# ENSO project

- sst.mnmean.nc is a netcdf file that you can open in python with the xarray library. It contains monthly mean sea surface temperature from the NOAA reanalysis\
- mslp_coarse.nc contains atmospheric pressure above the PAcific Ocean.\
- project_elnino.ipynb is a juyter notebook containing the project sibject/introduction.\
- elnino.ipynb is a jupyter notebook to answering to our problematic on the predactibility of El Nino phenomenon.

The goal of the project is to study existing classifiers. For that, we used the $\textit{Nino sst Indices}$ (https://climatedataguide.ucar.edu/climate-data/nino-sst-indices-nino-12-3-34-4-oni-and-tni)
